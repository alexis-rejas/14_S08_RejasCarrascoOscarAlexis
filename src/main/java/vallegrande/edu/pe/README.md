##  Descripción

Este proyecto es un sistema de biblioteca hecho en Java Swing.
Permite registrar libros, préstamos y usuarios usando ventanas.

---

##  Mejoras que hice

* Cambié los colores para que se vea más ordenado
* Agregué botones con iconos (➕ ✏️ 🗑️ ⬅)
* Mejoré la alineación de los formularios
* Puse un logo en la pantalla principal

---

##  Editar Préstamo

Ahora se puede editar un préstamo:

1. Seleccionas una fila
2. Presionas el botón editar
3. Te salen ventanas para cambiar:

    * libro
    * usuario
    * fecha
4. Se actualiza en la tabla

---

##  Validaciones

* No se puede dejar campos vacíos
* Si no seleccionas una fila sale mensaje
* Mensajes como:

    * "Complete todos los campos"
    * "Seleccione un registro"

---

##  Búsqueda (extra)

En libros agregué un buscador:

* Puedes buscar por título o autor
* Se filtra mientras escribes

---

##  Guardado (extra)

* Los libros se guardan en un archivo `.csv`
* Cuando vuelves a abrir el programa, los datos siguen ahí

---

##  Cómo ejecutar

1. Abrir en IntelliJ
2. Ejecutar `App.java`
3. Usar el menú

---

##  Cementario 

El sistema funciona con CRUD completo y usa el patrón MVC.
